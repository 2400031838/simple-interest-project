# Simple Interest Project

This repository contains files for the Simple Interest project assignment.

## Files included
- LICENSE (Apache 2.0)
- README.md
- CODE_OF_CONDUCT.md
- CONTRIBUTING.md
- simple-interest.sh

## How to run simple-interest.sh
1. Make the script executable using: `chmod +x simple-interest.sh`
2. Run the script with: `./simple-interest.sh`

The script calculates simple interest based on principal, rate, and time entered by the user.
